What Changes

Removed
  • 6 static image imports (EN + TR × 3 features)
  • ImageWithFallback component usage
  • Language-conditional image selection
  • aspect-[4/3] container
Added
  • LazyYouTube component with video ID per feature
  • aspect-video (16:9) container
  • Same videos for both languages (Turkish demos)
  • PostHog tracking on play
Key simplification: Since videos are Turkish-only, we no longer need language-conditional image imports. The 6 static image imports and the isTr ? trImage : enImage logic get replaced with 3 hardcoded YouTube video IDs. Less code, simpler component.
