Mobile Behavior

On mobile (< 768px), the 3-card grid stacks to a single column. Each card takes full width. Play buttons become larger (56px) for touch targets.

Interaction Flow

1. User sees thumbnail grid → clicks play on any card
2. Thumbnail fades out, YouTube iframe fades in with autoplay
3. PostHog fires video_played event with video ID + section
4. After watching, bridge CTA leads to Calculator section

Variant B difference: Same layout but bridge CTA uses rose-500 color instead of teal to match variant B's color scheme.
